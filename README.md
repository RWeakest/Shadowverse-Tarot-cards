# 影之诗 X 以撒 MOD

本mod还处于测试阶段，如果你发现了任何bug或者有任何建议，欢迎随时联系我。（本mod的翻译均为机翻，可能有不准确的地方）

## 模组内容

包含了8个被动道具，以影之诗《花醉游戏》卡包的7张塔罗牌虹卡为设计来源

* [漫步的《愚者》·琳库露](#item-the-fool)
* [脚踩天穹的《倒吊人》·罗弗拉德](#item-the-hanged-man)
* [转动的《命运之轮》·斯洛士](#item-wheel-of-fortune)
* [思念的《力量》·索菲娜](#item-strength)
* [充实的《恋人》·米路缇欧](#item-the-lovers)

### <a id="item-the-fool"></a>漫步的《愚者》·琳库露

| 类别 | 充能 | 品质 | 标签 | 道具池 |
| :---: | :---: | :---: | :---: | :---: |
| 被动 | - | 3 | 攻击性, 可召唤 | 宝箱房, 商店 |

**道具效果：**
* 清理房间时，有 30% 的概率生成一张卡牌《愚者》
* 玩家触发任何传送效果时，永久为当前房间内的所有敌人削减一点生命上限

---

### <a id="item-the-hanged-man"></a>脚踩天穹的《倒吊人》·罗弗拉德

| 类别 | 充能 | 品质 | 标签 | 道具池 |
| :---: | :---: | :---: | :---: | :---: |
| 被动 | - | 3 | 非贪婪模式, 攻击性 | 宝箱房, 商店 |

**道具效果：**
* 清理房间时，有 30% 的概率生成一张卡牌《倒吊人》
* 当玩家处于飞行状态，或以飞行状态进入房间时，获得“混沌”的效果（生成道具时随机抽取道具池）
* 玩家所拾取道具的每个“道具池”标签，提供 +0.15 伤害修正

---

### <a id="item-wheel-of-fortune"></a>转动的《命运之轮》·斯洛士

| 类别 | 充能 | 品质 | 标签 | 道具池 |
| :---: | :---: | :---: | :---: | :---: |
| 被动 | - | 3 | 攻击性, 可召唤 | 宝箱房, 商店 |

**道具效果：**
* 清理房间时，有 30% 的概率生成一张卡牌《命运之轮》
* 使用硬币时，将随机触发以下效果之一：
  * 掉落物：生成 1 硬币 / 生成 1 心 / 生成 1 炸弹 / 生成 1 钥匙 / 生成 1 随机掉落物
  * 移速加成： +0.02 / +0.01
  * 射程加成： +0.40 / +0.20 / +0.10
  * 射速修正： +0.30 / +0.20 / +0.05
  * 弹速加成： +0.03 / +0.02 / +0.01
  * 伤害修正： +0.5 / +0.1 / +0.02
  * 幸运加成： +1 / +1
  * 其他： 什么也不发生

### <a id="item-the-lovers"></a>充实的《恋人》·米路缇欧

| 类别 | 充能 | 品质 | 标签 | 道具池 |
| :---: | :---: | :---: | :---: | :---: |
| 被动 | - | 3 | 非游魂长子名分, 可召唤 | 宝箱房, 商店 |

**道具效果：**
* 清理房间时，有 30% 的概率生成一张卡牌《恋人》
* 房间内生成心类掉落物时，将其升级：
  * 魂心、半魂心：转化为 黑心
  * 红心、半红心： 转化为以下心类之一：
    * 70% 概率改为 双红心
    * 10% 概率改为 魂心
    * 6% 概率改为 黑心
    * 6% 概率改为 骨心
    * 6% 概率改为 金心
    * 2% 概率改为 永恒之心

### <a id="item-justice"></a>约束的《正义》·伊兰翠

| 类别 | 充能 | 品质 | 标签 | 道具池 |
| :---: | :---: | :---: | :---: | :---: |
| 被动 | - | 3 | 攻击性, 可召唤 | 宝箱房, 商店 |

**道具效果：**
* 清理房间时，有 30% 的概率生成一张卡牌《正义》
* 拾取基础掉落物（硬币、心、炸弹、钥匙）时，玩家获得0.8秒无敌
* 进入新的层级时，游戏会结算上一层未拾取的所有掉落物，并依据其价值转化为伤害修正（合成宝袋中掉落物价值总和 × 0.01）
